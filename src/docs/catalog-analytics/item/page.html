В данном аналитическом обзоре представлен сравнительный анализ различий стандартов СТО IIOGT S.10.1 2020 (стандарт отраслевой организации АНО «Институт нефтегазовых технологических инициатив», ГОСТ 32601, ISO 13709/API 610). Цель обзора дать возможность участникам АНО «Институт нефтегазовых технологических инициатив» понять различия вышеобозначенных стандартов с целью дальнейшего принятия решения по их применению.

Сопоставительный анализ различий технический требований не призван рассматриваться как дискредитацию какого-либо лица или стандарта, АНО «Институт нефтегазовых технологических инициатив» (ИНТИ) с большим уважением относиться ко всем лицам, делающих вклад в развитие стандартизации. Данный обзор носит информационный характер.
